# MVP v16 Release Notes

This build merges the raster-art prototype with an audio-enabled, release-oriented web build.

## Key updates

- Procedural prototype audio is included as WAV files.
- Runtime audio manager supports BGM, ambience, and SFX.
- Local progress is saved with `localStorage`.
- Users can mute, change volume, enter fullscreen, and reset progress.
- GitHub Pages relative paths are preserved.
- Mobile pointer/touch support is preserved.
- No runtime SVG files are included.

## Deployment

Upload the contents of this folder to GitHub Pages or any static hosting service. No build step and no server-side routing are required.
