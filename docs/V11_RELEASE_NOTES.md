# MVP v16 Release Notes

This build increases the visual density of the playable prototype.

## New in v11

- 20 additional close-up / insert shots derived from the high-quality v10 cinematic images.
- 1 cinematic contact sheet for QA review.
- Expanded cinematic queues:
  - Opening now has establishing shot, tide/foot insert, shell macro, hand-shell close-up.
  - Beach now has tide and letter close-ups.
  - Workshop now has artisan face, worktable, tools, shell fragments, hairpin macro.
  - Market now has lantern steam, boss close-up, spoon/porridge inserts.
  - Lion finale now has red shadow, drum silhouette, lion eye macro, final guardian close-up.
- The game now behaves more like a visual novel with staged camera cuts rather than a simple background + dialogue prototype.

## Deployment

Upload the contents of `haicheng-mvp-v16-text-polish` to GitHub Pages or any static host. No build step is required.
