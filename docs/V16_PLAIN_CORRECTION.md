# MVP v16 Correction

This correction removes the emotional scene one-liners from the HUD.

## Removed approach

The previous scene chip used poetic/emotional one-liners. That made the interface feel like it was telling the player what to feel.

## Current approach

The scene chip now uses neutral location/task information:

- Start: 开始
- Beach: 寻找旧信与贝壳
- Workshop: 询问工匠，修复发簪
- Market: 饮一碗热粥
- Lion: 终章预告

The intended emotion should be carried by:

- image composition
- character response
- camera cuts
- silence and audio
- interaction results
- pacing
