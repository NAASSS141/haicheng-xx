# MVP v16 Release Notes

This build turns the prototype into a cleaner public playtest package.

## New in v9

- Start / Continue menu.
- Help overlay with controls.
- Chapter progress panel.
- Keyboard shortcuts:
  - Enter: start from title menu
  - C: continue from title menu
  - H: hotspot debug
  - I: QA panel
  - M: mute
  - F: fullscreen
  - / or ?: help
- Improved localStorage save/resume behavior.
- Preserved GitHub Pages relative paths.
- Preserved mobile pointer/touch support.
- Preserved raster-only runtime image assets and packaged prototype audio.

## Static deployment

Upload the `haicheng-mvp-v16-text-polish` folder contents to GitHub Pages. No build step is required.
