# MVP v16 Release Notes

This build targets perceived quality, immersion, and release feel.

## New in v16

- Compact floating control dock instead of an always-open debug toolbar.
- Scene chip with chapter title + mood line.
- Pointer parallax across background, scene layers, and characters.
- Soft fade overlay for scene transitions.
- Audio crossfade for BGM and ambience.
- Optional vibration feedback for chapter transitions and puzzle completion.
- Progress panel restyled into a cleaner pill HUD.
- Inventory button restyled into a cleaner floating pill.

## Preserved

- Flowfix mainline logic from v15.
- Immersive raster post-process overlays.
- Gallery / credits / ending replay buttons.
- Character performance dialogue layer.
- Cinematic still library.
- Save/resume, GitHub Pages static deployment, and mobile touch support.
- Zero runtime SVG files.

## Deployment

Upload the contents of `haicheng-mvp-v16-text-polish` to GitHub Pages. No build step is required.
