# 海城 MVP v16 发布前 QA 清单

## 1. 基础运行

- [ ] 打开 `index.html` 无空白页。
- [ ] 开场贝壳可点击。
- [ ] 预加载界面完成后进入游戏。
- [ ] 无控制台语法错误。

## 2. 主流程

- [ ] 开场进入海边。
- [ ] 未读信时点击贝壳提示“先看看信”。
- [ ] 读信后点击贝壳进入工坊。
- [ ] 工坊第一次点击工匠出现首次对话。
- [ ] 第二次点击工匠解锁工作台。
- [ ] 工作台拼贝壳完成进入夜粥街。
- [ ] 夜粥街点击凳子后解锁粥锅。
- [ ] 做粥完成后进入鼓狮预告。
- [ ] 结尾出现“未完待续”。

## 3. 分镜与演出

- [ ] 对白出现时分镜暂停。
- [ ] 对白点完后分镜继续。
- [ ] 白光/黑场转场可见。
- [ ] overlay 图像不会挡死主操作。
- [ ] 镜头动作无明显抖动错误。

## 4. GitHub Pages

- [ ] `node tools/check-github-pages.mjs` 通过。
- [ ] 没有 `/assets/...` 绝对路径。
- [ ] 没有服务端路由依赖。
- [ ] 部署到仓库子路径仍能加载 `./game.js` 与 `./styles.css`。

## 5. 素材

- [ ] `node tools/check-assets.mjs` 通过。
- [ ] `node tools/check-naming.mjs` 通过。
- [ ] QA 面板 missing loaded assets 为 0。
- [ ] 正式图替换后没有变形。

## 6. 移动端

- [ ] 横屏可正常玩。
- [ ] 竖屏显示横屏提示。
- [ ] 所有热点可点击。
- [ ] `node tools/check-touch-targets.mjs` 没有高风险提示。

## 7. 音频

- [ ] 未加入正式音频前，游戏不会因音频缺失报错。
- [ ] 加入正式音频后填写 `assets/audio/licenses/audio_license_registry.csv`。
- [ ] 只使用 CC0/Public Domain/明确免费商用且无需二次授权的音频。
