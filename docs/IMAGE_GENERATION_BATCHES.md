# 海城 MVP v16 图像生成批次表

本表用于安排图像生成，不建议一次性生成全部素材。先做能决定风格的主画面，再补人物、道具、特效。

## Batch 01：五张主画面

1. `bg_start_shell_scene.jpg`：黑蓝湿沙上的微光贝壳。
2. `bg_beach_dusk_full.jpg`：黄昏海边、防浪堤、湿沙、远船灯。
3. `bg_workshop_full.jpg`：老贝壳工坊、木桌、贝壳碎片、窗外海光。
4. `bg_market_full.jpg`：夜粥街、粥锅蒸汽、湿润路面、暖灯。
5. `bg_lion_dark.jpg`：黑暗中醒狮眼部预告。

验收标准：即使不显示任何对白，也能看出这是一个有电影感的互动叙事游戏，而不是 H5 页面。

## Batch 02：角色差分

### 主角

- 待机：安静站在海边。
- 看信：低头展开旧信。
- 伸手：触碰贝壳。

### 老工匠

- 背对打磨。
- 转头看主角。

### 老板娘

- 忙碌待机。
- 切鸭。
- 递照片/递物。

验收标准：角色和场景光源一致，不像贴纸；人物边缘干净，透明背景正常。

## Batch 03：谜题与物件

- 发光贝壳。
- 旧信展开。
- 五片贝壳拼图。
- 完整贝纹发簪。
- 旧木盒打开。
- 工坊旧照片。
- 空碗、粥底、鸭肉、清汤、葱花、酱料、完成夜粥。

验收标准：每个物件都能作为点击热点或谜题元素单独使用。

## Batch 04：动态与转场特效

- 贝壳珍珠光。
- 海浪泡沫。
- 工坊尘埃。
- 工坊窗光。
- 夜市蒸汽。
- 红绸飞过。
- 狮眼光。
- 回忆颗粒。

验收标准：特效透明背景或可叠加，不遮挡主体，不像网页装饰贴片。

## Batch 05：正式发布前补齐

- 所有 SVG 占位层换成高质量 PNG/WebP，除非 SVG 本身只是调试用。
- 生成压缩 WebP 版本。
- 运行命名与路径检查。
- 把替换记录写入 `qa/asset_replacement_log.csv`。
